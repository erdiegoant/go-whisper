# Phase 16 — First-launch Onboarding (Pro)

> **Goal:** A new Pro user goes from DMG to first transcription in under 2 minutes.

On first launch (detected via absence of `~/.config/gowhisper/config.yaml`):

1. Welcome screen — brief explanation of what GoWhisper does
2. Model selection — tiny / small / medium with size and speed notes; download starts
   in-app with a progress bar (no terminal needed)
3. Permissions guide — step-by-step for Microphone and Accessibility access with
   screenshots
4. Hotkey confirmation — show the default ⌥Space hotkey, allow changing it immediately
5. Done — app lands in the menubar ready to use
